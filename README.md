# REAP Odyssey - Vercel Deployment

A single-page application showcasing the REAP compression journey for GLM-4.7.

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deploy to Vercel

1. Push this repository to GitHub
2. Import project in Vercel
3. Deploy!

The app is configured for static export and will be served as a static site.
